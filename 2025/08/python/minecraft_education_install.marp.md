# 教育版マインクラフト<br>インストール手順

---

## 教育版マインクラフトとは？

**Minecraft Education Edition**は、教育機関向けに開発されたMinecraftの特別版です。

### 主な特徴
- 教育機関向けの機能が充実
- Pythonコードを実行できる
- 無料で利用可能（教育機関向け）
- 教師向けの管理機能

---

## ダウンロード

### 公式サイトからダウンロード

1. **Minecraft Education Edition公式サイト**にアクセス
   - URL: https://education.minecraft.net/

2. **ダウンロード**をクリック

3. **Windows版**を選択

---

### ダウンロード手順

1. 公式サイトの「ダウンロード」ページを開く
2. 「Windows 10/11」を選択
3. 「ダウンロード」ボタンをクリック
4. インストーラー（`.exe`ファイル）がダウンロードされます

---

## Windowsへインストール

### インストール手順（1/2）

1. **ダウンロードしたインストーラーを実行**
   - `.exe`ファイルをダブルクリック

2. **インストールウィザードに従う**
   - 「次へ」をクリック
   - インストール先を選択（デフォルトでOK）

---

### インストール手順（2/2）

3. **インストールを開始**
   - 「インストール」をクリック
   - インストールが完了するまで待機

4. **完了**
   - 「完了」をクリック
   - デスクトップにショートカットが作成されます

---

## サインイン

### サインイン手順

1. **Minecraft Education Editionを起動**

2. **「サインイン」をクリック**

3. **Microsoftアカウントでサインイン**
   - 提供されたアカウントを入力
   - パスワードを入力

4. **サインイン完了**
   - ホーム画面が表示されます

## 基本操作

### 移動操作

- **W**: 前進
- **S**: 後退
- **A**: 左移動
- **D**: 右移動
- **スペース**: ジャンプ
- **Shift**: しゃがむ

---

### マウス操作

- **マウス移動**: 視点を変更
- **左クリック**: ブロックを破壊
- **右クリック**: ブロックを配置
- **ホイール**: アイテムを選択

---

### その他の操作

- **E**: インベントリを開く
- **Tab**: チャットを開く
- **Esc**: メニューを開く
- **F3**: デバッグ情報を表示

---

## Python実行手順

### コードエディタを開く

1. **ゲーム内で「C」キーを押す**
   - または、メニューから「コードエディタ」を選択

2. **「Python」を選択**
   - コードエディタが開きます

---

### Pythonコードの作成

1. **コードエディタでコードを入力**

```python
def on_on_chat():
    blocks.place(blocks.WOOL, world(0, 0, 0))
player.on_chat("run", on_on_chat)
```

2. **「実行」ボタンをクリック**
   - または、チャットで`run`と入力

---

### Pythonコードの実行

1. **チャットでコマンドを入力**
   - チャットを開く（Tabキー）
   - `run`と入力してEnter

2. **コードが実行される**
   - ブロックが配置されます
   - エラーがある場合は、エラーメッセージが表示されます

---

### よく使うPython API

```python
# ブロックを配置
blocks.place(blocks.WOOL, world(x, y, z))

# プレイヤーの位置を取得
pos = player.position()

# チャットにメッセージを表示
player.say("Hello!")
```

---

### ファイルのインポート

1. **コードエディタで「インポート」をクリック**

2. **Pythonファイル（`.py`）を選択**

3. **ファイルが読み込まれます**
   - コードエディタに表示されます
   - 実行可能になります

## トラブルシューティング

### よくある問題

**Q: サインインできない**
- 学校提供のアカウントを使用しているか確認
- インターネット接続を確認

**Q: Pythonコードが実行されない**
- コードエディタが開いているか確認
- エラーメッセージを確認

---

### トラブルシューティング（続き）

**Q: ブロックが配置されない**
- 座標が正しいか確認
- プレイヤーの位置を確認

**Q: ファイルがインポートできない**
- ファイル形式が`.py`か確認
- ファイルサイズが大きすぎないか確認
